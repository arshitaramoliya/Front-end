# Frontend Basics — HTML, CSS & JavaScript

A collection of foundational frontend exercises covering static web pages, basic styling, and JavaScript DOM interaction.

## Contents

| File / Folder | Description |
|---|---|
| `index.html` | Main entry page |
| `css.html` | CSS styling experiments |
| `one.html` | Structural HTML layout practice |
| `data.js` | DOM utility — `displaydata()` function for element selection |
| `static-web/` | Static multi-page website exercises |
| `arsh/` | Additional HTML/CSS practice files |

## Concepts Covered

- HTML page structure and semantic elements
- CSS styling and layout basics
- JavaScript DOM selection with `document.getElementById` / `querySelector`
- Organising a multi-page static site
